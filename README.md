# 🎯 MERN Quiz App

An **interactive quiz application** built with the **MERN stack** (MongoDB, Express, React, Node.js).  
Take quizzes, get instant results, and track your performance!  

---

## 🛠️ Features

- 🏆 Multiple-choice quizzes  
- 📊 Instant score and feedback  
- 🎨 Clean and responsive UI using Tailwind CSS  
- ⚡ Fast and lightweight
